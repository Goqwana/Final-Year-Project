# Hybrid MoM-PO Solver for Antenna Placement Analysis

**Final Year Project (Skrispie)**  
Stellenbosch University | Electrical & Electronics Engineering

## Project Overview
Formulated the scattered problem using integral equations.  
Implemented Physical Optics (PO) and Method of Moments (MoM) solvers in MATLAB.  
Developed a Hybrid MoM-PO solver to solve the radiated magnetic field.  
Validated with known solutions and experimental data using MATLAB & FEKO.  
Verified near-field and far-field patterns.
