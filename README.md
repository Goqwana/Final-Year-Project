# Engineering Projects – Odwa Goqwana

## Hybrid MoM-PO Solver for Antenna Placement Analysis
**Module:** Final Year Project (Skrispie) • 6 Months

- Formulated the scattered problem using integral equations.
- Implemented Physical Optics (PO) and Method of Moments (MoM) solvers for plane wave scattering in MATLAB.
- Developed a Hybrid MoM-PO solver in MATLAB to solve the radiated magnetic field.
- Validated the solver with known solutions and experimental data using MATLAB & FEKO.
- Verified near-field and far-field patterns by comparing MATLAB with FEKO.
